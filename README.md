## 📋 About the Project

clemantcarlos.github.io is my older personal portfolio site, deployed via GitHub Pages. It showcases my education background, projects, and professional links. Built with vanilla HTML, CSS, and JavaScript with jQuery for interactivity.

## ✨ Features

| Feature | Description |
|---------|-------------|
| Responsive Design | Works across different screen sizes |
| jQuery Interactivity | Dynamic UI interactions |
| Hover.css Animations | CSS hover effects and transitions |
| FontAwesome Icons | Rich icon set for social links |
| GitHub Pages | Free hosting via GitHub |

## 📂 Project Structure

```
clemantcarlos.github.io/
├── index.html          # Main portfolio page
├── css/                # Stylesheets
├── js/                 # JavaScript files
├── img/                # Images and assets
├── package.json        # Dependencies (hover.css)
```

## 🛠️ Tech Stack

- **Markup:** HTML5
- **Styling:** Custom CSS with Hover.css
- **JavaScript:** Vanilla JS + jQuery
- **Icons:** FontAwesome
- **Hosting:** GitHub Pages

## 🚀 Getting Started

### Prerequisites

- A modern web browser
- (Optional) Prepros for SCSS compilation

### Installation

```bash
# Clone the repository
git clone https://github.com/clemantcarlos/clemantcarlos.github.io.git
cd clemantcarlos.github.io

# Install dependencies
npm install
```

### Running Locally

Simply open `index.html` in your browser or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve .
```
